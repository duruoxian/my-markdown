## 标题

### 使用 `#` 符号表示标题

```markdown
# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题
```

**效果：**

# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

> **注意：** `#` 后面必须有一个空格

## 文本格式

### 1. 加粗文本

```markdown
**加粗文本**
__加粗文本__（两种方法都可以）
```

**效果：** **加粗文本**

### 2. 斜体文本

```markdown
*斜体文本*
_斜体文本_（两种方法都可以）
```

**效果：** *斜体文本*

### 3. 加粗 + 斜体

```markdown
***加粗斜体***
___加粗斜体___
**_加粗斜体_**
```

**效果：** ***加粗斜体***

### 4. 删除线

```markdown
~~删除的文本~~
```

**效果：** ~~删除的文本~~

### 5. 下标和上标（某些 Markdown 编辑器支持）

```markdown
H~2~O
E=mc^2^
```

**效果：** H~2~O 和 E=mc^2^

### 6. 高亮标记（某些编辑器支持）

```markdown
==高亮文本==
```

**效果：** ==高亮文本==

## 列表

### 1. 无序列表

```markdown
- 项目 1
- 项目 2
- 项目 3
```

**效果：**
- 项目 1
- 项目 2
- 项目 3

**也可以用 `*` 或 `+`：**

```markdown
* 项目 1
* 项目 2

+ 项目 1
+ 项目 2
```

### 2. 有序列表

```markdown
1. 第一项
2. 第二项
3. 第三项
```

**效果：**
1. 第一项
2. 第二项
3. 第三项

### 3. 嵌套列表

```markdown
- 一级项目 1
  - 二级项目 1.1
  - 二级项目 1.2
- 一级项目 2
  - 二级项目 2.1
    - 三级项目 2.1.1
```

**效果：**
- 一级项目 1
  - 二级项目 1.1
  - 二级项目 1.2
- 一级项目 2
  - 二级项目 2.1
    - 三级项目 2.1.1

### 4. 混合列表

```markdown
1. 有序项目 1
2. 有序项目 2
   - 无序子项目 1
   - 无序子项目 2
3. 有序项目 3
```

## 链接和图片

### 1. 基本链接

```markdown
[链接文字](https://example.com)
```

**效果：** [链接文字](https://example.com)

### 2. 带标题的链接

```markdown
[链接文字](https://example.com "鼠标悬停显示的标题")
```

**效果：** [链接文字](https://example.com "鼠标悬停显示的标题")

### 3. 自动链接

```markdown
<https://example.com>
<user@example.com>
```

**效果：**
<https://example.com>
<user@example.com>

### 4. 引用式链接

```markdown
这是一个[链接][1]，这是另一个[链接][2]。

[1]: https://example.com
[2]: https://github.com
```

### 5. 内部锚点链接

```markdown
[跳转到标题](#标题)
```

点击会跳转到页面中的 `## 标题` 部分

### 6. 基本图片

```markdown
![图片描述](图片URL)
```

**例如：**
```markdown
![GitHub Logo](https://github.com/fluidicon.png)
```

### 7. 带链接的图片

```markdown
[![图片描述](图片URL)](链接URL)
```

这样点击图片会跳转到链接。

### 8. 调整图片大小（HTML 方法）

```markdown
<img src="图片URL" alt="描述" width="200" height="200">
```

## 代码

### 1. 行内代码

```markdown
这是一段 `行内代码` 示例。
```

**效果：** 这是一段 `行内代码` 示例。

### 2. 代码块（缩进方法）

```markdown
    // 这是代码块
    function hello() {
        console.log("Hello");
    }
```

> **注意：** 开头缩进 4 个空格或 1 个 Tab

### 3. 代码块（围栏方法 - 推荐）

```markdown
​```
这是代码块
无需缩进
​```
```

### 4. 带语言高亮的代码块

```markdown
​```cpp
#include <iostream>

int main() {
    std::cout << "Hello, World!" << std::endl;
    return 0;
}
​```
```

**效果：**

```cpp
#include <iostream>

int main() {
    std::cout << "Hello, World!" << std::endl;
    return 0;
}
```

### 5. 其他编程语言示例

```markdown
​```python
def hello():
    print("Hello, World!")
​```
```

```markdown
​```javascript
function hello() {
    console.log("Hello, World!");
}
​```
```

```markdown
​```java
public class Main {
    public static void main(String[] args) {
        System.out.println("Hello, World!");
    }
}
​```
```

```markdown
​```bash
echo "Hello, World!"
​```
```

**支持的语言：** `python`, `java`, `cpp`, `c`, `javascript`, `html`, `css`, `sql`, `bash`, `shell`, `powershell`, `ruby`, `go`, `rust`, 等等

## 表格

### 基本表格

```markdown
| 表头 1 | 表头 2 | 表头 3 |
|--------|--------|--------|
| 内容 1 | 内容 2 | 内容 3 |
| 内容 4 | 内容 5 | 内容 6 |
```

**效果：**

| 表头 1 | 表头 2 | 表头 3 |
|--------|--------|--------|
| 内容 1 | 内容 2 | 内容 3 |
| 内容 4 | 内容 5 | 内容 6 |

### 表格对齐

```markdown
| 左对齐 | 居中对齐 | 右对齐 |
|:-------|:--------:|--------:|
| 左 | 中 | 右 |
| 左 | 中 | 右 |
```

**效果：**

| 左对齐 | 居中对齐 | 右对齐 |
|:-------|:--------:|--------:|
| 左 | 中 | 右 |
| 左 | 中 | 右 |

**对齐说明：**
- `:---` 左对齐
- `:---:` 居中对齐
- `---:` 右对齐
- `---` 默认对齐

### 复杂表格示例

```markdown
| 语言 | 扩展名 | 描述 |
|:---:|:-----:|------|
| C++ | .cpp | 编译型语言 |
| Python | .py | 解释型语言 |
| JavaScript | .js | Web 脚本语言 |
```

**效果：**

| 语言 | 扩展名 | 描述 |
|:---:|:-----:|------|
| C++ | .cpp | 编译型语言 |
| Python | .py | 解释型语言 |
| JavaScript | .js | Web 脚本语言 |

## 高级用法

### 1. 转义特殊字符

如果你想显示 Markdown 特殊符号本身，使用反斜杠 `\`：

```markdown
\# 不是标题
\* 不是列表
\` 不是行内代码
```

**效果：**
\# 不是标题
\* 不是列表
\` 不是行内代码

### 2. HTML 元素

在 Markdown 中可以直接使用 HTML：

```markdown
<div style="color: red;">
这是红色文字
</div>
```

### 3. 脚注（某些编辑器支持）

```markdown
这是一个脚注[^1]

[^1]: 脚注内容
```

### 4. 数学公式（某些编辑器支持）

```markdown
行内数学：$E = mc^2$

公式块：
$$
\frac{-b \pm \sqrt{b^2 - 4ac}}{2a}
$$
```

### 5. 注释（不会显示）

```markdown
<!-- 这是注释，不会显示在最终结果中 -->
```

### 6. 折叠内容（HTML）

```markdown
<details>
<summary>点击展开</summary>

这是折叠的内容
可以包含多行

甚至可以包含代码：
```cpp
int main() {}
```

</details>
```

**效果：**

<details>
<summary>点击展开</summary>

这是折叠的内容
可以包含多行

甚至可以包含代码：
```cpp
int main() {}
```

</details>

### 7. 警告框（GitHub 特定）

```markdown
> [!NOTE]
> 这是一个提示框

> [!WARNING]
> 这是一个警告框

> [!IMPORTANT]
> 这是一个重要信息

> [!CAUTION]
> 这是一个注意框

> [!TIP]
> 这是一个贴士框
```

## 总结

Markdown 是一种轻量级标记语言，具有以下优点：

✅ **易学易用** - 语法简单明了
✅ **纯文本** - 任何编辑器都可以编辑
✅ **可转换** - 可以转换为 HTML、PDF 等格式
✅ **广泛支持** - GitHub、Reddit、Discord 等都支持

现在你已经掌握了 Markdown 的所有基础用法和高级技巧，可以开始写你的文档了！🚀
